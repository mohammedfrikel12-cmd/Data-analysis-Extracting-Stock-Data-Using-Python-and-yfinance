# Extracting Stock Data Using Python and yfinance

This project demonstrates how to extract and analyze stock market data using Python and the `yfinance` library. The notebook focuses on retrieving financial information for companies such as Apple and AMD, including company details, historical share prices, and dividend data.

The main goal of this project is to practice working with financial APIs/libraries, handling JSON data, using Pandas DataFrames, and visualizing stock data with Python.

## What I Learned

Through this project, I practiced:

- Using the `yfinance` library to access stock market data
- Creating ticker objects for specific companies
- Extracting company information from JSON files
- Working with historical stock price data
- Understanding stock data columns such as Open, High, Low, Close, and Volume
- Using Pandas to inspect and manipulate financial datasets
- Visualizing stock prices and dividends with Matplotlib
- Understanding how Python can be used in financial data analysis

## Technologies Used

- Python
- yfinance
- Pandas
- Matplotlib
- JSON
- Jupyter Notebook

## Project Overview

The notebook includes examples of how to:

1. Install and import the required libraries
2. Create a stock ticker object using `yfinance`
3. Extract company information
4. Load and read JSON files
5. Retrieve historical share price data
6. Convert stock data into a Pandas DataFrame
7. Plot historical stock prices
8. Extract and visualize dividend data
